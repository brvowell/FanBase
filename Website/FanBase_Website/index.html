Project Goals

The primary function of FanBase is to help users find or create and plan tailgates for friends and family. By using FanBase, users can connect and share information in-app to work towards a flawless tailgate experience for all involved.

Tailgates Near Me

Users can create tailgates of their own as well as view the details of other public tailgates around. If a user creates a private tailgate then he or she must send invites to other users in order for them to join the tailgate and view its contents. Users will also be able to use the map to select a spot for reaccuring tailgates.

In-App Messaging

As users join tailgates, they will also be added to a group message inside of FanBase allowing for quick communication for everyone attending the tailgate. If a user has reaccuring tailgates, FanBase will keep track of everyone attending each event.

Gamet-time Notifications

FanBase will use the user's location to send all users in tailgate a notification for when the game is about to start
